# Cmux Remote 1.0.4 Release Notes

## App Store “What’s New” — English

Cmux Remote 1.0.4 improves terminal responsiveness and prepares the renderer for richer terminal styling.

- Speeds up terminal rendering by caching parsed rows and drawing style runs instead of reprocessing every cell on each frame.
- Keeps a larger bounded terminal history during subscribe and refresh, reducing abrupt 24-line refreshes and unnecessary full-screen reloads.
- Adds 256-color and true-color ANSI rendering support for relay data that includes ANSI styles, while preserving Korean/CJK and emoji column alignment.
- Improves checksum reconciliation so styled rows do not trigger redundant full refreshes.
- Verified on a physical iPhone against a live cmux relay connection.

## App Store “새로운 기능” — Korean

cmux Remote 1.0.4는 터미널 화면 반응성을 높이고 더 풍부한 터미널 스타일 렌더링을 위한 기반을 준비한 업데이트입니다.

- 매 프레임 모든 셀을 다시 처리하지 않도록 파싱된 row와 style run을 캐시해 터미널 렌더링 속도를 개선했습니다.
- 구독/새로고침 시 120줄 bounded history를 유지해 갑작스러운 24줄 축소와 불필요한 전체 화면 reload를 줄였습니다.
- ANSI 스타일 데이터가 relay에서 제공될 때 256색/truecolor ANSI 렌더링을 지원하며, 한국어/CJK/이모지 컬럼 정렬을 유지합니다.
- 스타일이 포함된 row가 불필요한 전체 새로고침을 유발하지 않도록 checksum reconcile을 개선했습니다.
- 실제 iPhone에서 live cmux relay 연결로 검증했습니다.

## Known limitations

- Exact desktop cmux color fidelity still depends on cmux / relay exposing styled terminal rows. Live `surface.read_text` checks currently return plain text, so the iOS renderer is ready for ANSI styles but cannot invent colors that are not sent.
- Notifications are still local-only; APNs remains planned for v1.1.
