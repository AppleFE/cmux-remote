tutorial
01
Mac에서 cmux와 Tailscale을 켭니다.
같은 tailnet에 있어야 합니다.
02
Mac 터미널에서 릴레이를 실행합니다.
swift run cmux-relay serve --config ~/.cmuxremote/relay.json
03
relay.json의 listen을 열어둡니다.
실기기 연결은 0.0.0.0:80 또는 Tailscale IP 바인딩 필요.
04
이 웹 페이지를 릴레이 주소로 엽니다.
host:port는 현재 연결된 릴레이입니다.
05
Workspaces가 보이면 완료.
실패 시 Mac 릴레이/Tailscale 상태 확인.
